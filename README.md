# Autonomous Terrain Data Rover (ATDR)

An autonomous over-terrain rover designed for sensor-based data collection, obstacle detection, and object interaction in constrained environments.  
The system integrates mechanical design, embedded control, and real-world prototyping.

## Key Features
- Omni-wheel drivetrain for maneuverability over uneven terrain
- Ultrasonic sensors for obstacle detection
- Servo-driven vertical actuator for object interaction
- Embedded control using Arduino and C/C++
- Modular mechanical and electrical design

---

## Technologies Used
- **Programming:** C/C++, Arduino
- **Mechanical Design:** CAD (SolidWorks / Fusion-style workflow)
- **Electronics:** Ultrasonic sensors, motor drivers, servos
- **Fabrication:** Laser-cut chassis, 3D-printed components
