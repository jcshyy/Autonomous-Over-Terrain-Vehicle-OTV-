# Autonomous Terrain Data Rover (ATDR)

An autonomous over-terrain robotic platform designed for sensor-based data collection, obstacle detection, and object interaction in constrained or uneven environments.  
This project integrates mechanical design, embedded systems, and real-world prototyping.

---

## System Overview
![Isometric CAD](OTV%20IMAGES/cad_isometric_view.jpg)

The rover features an omni-wheel drivetrain for high maneuverability, a modular chassis for rapid iteration, and a vertically actuated end-effector for interaction with the environment.

## Key Features
- Omni-wheel drivetrain enabling lateral and rotational maneuverability
- Ultrasonic sensors for obstacle detection and spatial awareness
- Servo-driven vertical actuator for object interaction
- Modular mechanical and electrical architecture
- Designed for rapid prototyping and iterative testing

---

## Technologies Used
- **Programming:** C/C++, Arduino
- **Mechanical Design:** CAD (SolidWorks / Fusion-style workflow)
- **Electronics:** Ultrasonic sensors, motor drivers, servos
- **Fabrication:** Laser-cut chassis components, 3D-printed parts
